 # 🌿 Psicóloga Angélica Ferreira — Landing Page Profissional


Projeto de alta performance desenvolvido para a área de Psicologia Clínica, com foco em acessibilidade, design minimalista e conversão direta via WhatsApp.


## 🚀 Tecnologias Utilizadas


* **HTML5 Semântico:** Estruturação focada em SEO e acessibilidade.

* **CSS3 Custom Properties:** Sistema de design baseado em variáveis para fácil manutenção de cores e temas.

* **Vanilla JavaScript:** Implementação de animações de scroll (Reveal API) e interatividade sem dependências externas.

* **Responsividade Full:** Layout adaptável para dispositivos móveis, tablets e desktop (Mobile First).


## 🛠️ Diferenciais Técnicos


* **Performance:** Código limpo e otimizado para carregamento rápido em redes 4G/5G.

* **UX (User Experience):** Navegação fluída com scroll suave e call-to-action (CTA) estratégico.

* **Integração Lúdica:** Seção dedicada à **Ludoterapia (TCC Infantil)**, comunicando especialidade clínica de forma clara para os pais.

* **Clean Code:** Organização de pastas e nomenclatura de classes seguindo boas práticas.
